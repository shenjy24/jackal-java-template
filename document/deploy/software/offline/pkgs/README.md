# Offline Packages

Put offline installation packages in this directory before running `../install-offline-software.sh`.

Expected default file names:

- `git-binaries.linux-64bit.tar.gz`
- `docker-29.5.3.tgz`
- `docker-compose-linux-x86_64`

You can override these names with `GIT_ARCHIVE`, `DOCKER_ARCHIVE`, and `DOCKER_COMPOSE_BINARY`.
