# 离线安装包目录

执行 `../install-software.sh` 前，请先将离线安装包放到当前目录。

默认文件名如下：

- `git-binaries.linux-64bit.tar.gz`
- `docker-29.5.3.tgz`
- `docker-compose-linux-x86_64`

如需使用其他文件名，可通过 `GIT_ARCHIVE`、`DOCKER_ARCHIVE`、`DOCKER_COMPOSE_BINARY` 环境变量覆盖。
